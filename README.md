# 네이버 블로그 파일로 저장


[테스트 버전]<br>
    
    파이선 : 3.7.6
    beautifulsoup4 : 4.9.0
    lxml : 4.5.0
    requests : 2.23.0


[필요 패키지 설치]<br>

    $ pip install beautifulsoup4
    $ pip install lxml
    $ pip install requests


[실행 방법]<br>

    python .\download_naver_blog.py [네이버 블로그 링크] [저장 파일 이름]
    
    예 > 
    python .\download_naver_blog.py https://blog.naver.com/chandong83/221951781607 blog.html
